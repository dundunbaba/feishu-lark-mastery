# 知识库维护规则

**版本**：1.0.0
**最后更新**：2026-03-25

---

## 统一更新规则

### 官方知识（`模块/official/`）

| 项目 | 规则 |
|------|------|
| 来源 | 仅 open.feishu.cn / feishu.cn/hc |
| 触发 | 定期同步 / 操作报错发现过时 / 在线搜索后沉淀 |
| 格式 | 标注 `[官方]` + 链接 + 更新日期 |
| 禁止 | 不掺入任何经验内容 |

### 经验知识（`模块/experience/`）

| 项目 | 规则 |
|------|------|
| 来源 | Agent 自身操作 |
| 触发 | 每次操作成功/失败后 |
| 格式 | `### [日期] 标题` + 场景/问题/原因/解决/教训 |
| 禁止 | 不包含官方文档的大段复制 |

### 速查表（`模块/CHEATSHEET.md`）

| 项目 | 规则 |
|------|------|
| 来源 | 从 official + experience 提炼 |
| 触发 | 经验积累到一定量时 / 发现高频查询点 |
| 格式 | 表格为主，极简 |
| 原则 | 80% 问题在这解决 |

### 场景方案（`模块/scenarios/`）

| 项目 | 规则 |
|------|------|
| 来源 | 第三方教程 + 实战项目 |
| 触发 | 用户提供 / 项目完成后 |
| 格式 | 标注来源 + 适用条件 |
| 审核 | 需用户确认后���入 |

## 同步周期

| 动作 | 频率 |
|------|------|
| 官方文档检查 | 每月 1 次 |
| 经验沉淀 | 每次操作后 |
| CHEATSHEET 更新 | 积累 5+ 条经验后 |
| 知识健康度检查 | 每月 1 次 |
