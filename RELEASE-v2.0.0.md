## 📋 常见问题分类

### 权限问题
- 缺失权限：消息发送权限、多维表格读写权限
- 解决方案：提供权限申请链接和流程指导

### API 使用问题
- 不会使用：参数格式错误、API 路径错误
- 解决方案：提供速查表和示例代码

### 飞书官方未开发
- 功能缺失：某细节控制功能无 API
- 解决方案：推荐替代方案或提交需求给飞书

### 用户需求不清
- 需求模糊：目标不明确、有矛盾
- 解决方案：需求澄清引导流程
