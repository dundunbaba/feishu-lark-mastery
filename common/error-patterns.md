## 跨模块通用错误

| 错误码 | 含义 | 解决方案 |
|--------|------|---------|
| 99991400 | HTTP 频率限制 | 降低请求频率，指数退避重试 |
| 99991668 | Token 无效/过期 | 刷新 access_token |
| 99991672 | 权限不足 | 检查 API 权限是否已开启 |

## 错误排查流程

```
收到错误
  ↓
1. 先看本文件的通用错误
  ↓ 没找到
2. 去对应模块的 official/ 查模块错误码
  ↓ 没找到
3. 去 open.feishu.cn 搜索错误码
  ↓
4. 找到答案 → 补充到知识库
```
