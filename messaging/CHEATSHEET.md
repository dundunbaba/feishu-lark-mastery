## 发送消息（通过 OpenClaw message 工具）

### 基本用法

| 场景 | 方法 |
|------|------|
| 回复当前会话 | 直接回复（自动路由） |
| 主动发私聊 | `message(action=send, target="user:ou_xxx")` |
| 发群消息 | `message(action=send, target="chat:oc_xxx")` |

### @人语法

| 场景 | 格式 |
|------|------|
| 文本消息@某人 | `<at user_id='ou_xxx'>名字</at>` |
| 卡片消息@某人 | `<at id='ou_xxx'>名字</at>` |
| @所有人 | `<at user_id='all'>所有人</at>` |

## 消息类型

| msg_type | 说明 | content 格式 |
|----------|------|-------------|
| text | 文本 | `{"text": "内容"}` |
| post | 富文本 | 嵌套结构 |
| image | 图片 | `{"image_key": "xxx"}`（需先上传） |
| file | 文件 | `{"file_key": "xxx"}`（需先上传） |
| interactive | 卡片 | 卡片 JSON |
| share_chat | 群名片 | `{"chat_id": "oc_xxx"}` |

## 限制

- 单聊：5 QPS
- 群聊：5 QPS（群内机器人共享）
- 文本消息：最大 150KB
- 卡片/富文本：最大 30KB
- 图片需先上传 → `im/v1/image/create` → 拿 image_key
- 文件需先上传 → `im/v1/file/create` → 拿 file_key
- 不能用云文档上传素材接口的 file_token 发消息

## 常见错误码

| 错误码 | 含义 | 解决方案 |
|--------|------|---------|
| 230002 | 机器人不在群里 | 先拉机器人入群 |
| 230006 | 未启用机器人能力 | 开发者后台开启 |
| 230013 | 用户不在可用范围/已离职 | 配置应用可用范围 |
| 230020 | 触发频率限制 | 降频重试 |
| 230022 | 消息含敏感信息 | 修改消息内容 |
| 230025 | 消息体过大 | 文本<150KB，卡片<30KB |
| 230035 | 无发送权限 | 检查群禁言/用户屏蔽 |
| 230053 | 用户停止接收机器人消息 | 无法主动发单聊 |
